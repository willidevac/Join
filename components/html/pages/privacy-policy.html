Datenschutzerklärung

1. Verantwortliche Stelle

Verantwortlich für diese Lernanwendung ist das Join-Projektteam. Die Anwendung wird im Rahmen eines Ausbildungsprojekts entwickelt.

2. Welche Daten verarbeitet werden

Für die Registrierung werden Name, E-Mail-Adresse und Passwort abgefragt. Das Passwort darf nicht direkt in Firestore gespeichert werden, sondern muss über den verwendeten Authentifizierungsdienst verarbeitet werden.

3. Zweck der Verarbeitung

Die Daten werden verwendet, um ein Benutzerkonto anzulegen, den angemeldeten Nutzer zu erkennen und den Namen innerhalb der App anzeigen zu können.

4. Speicherung

Benutzerbezogene Daten werden nur so lange gespeichert, wie sie für die Nutzung der App oder fuer Testzwecke im Projekt erforderlich sind.

5. Weitergabe an Dritte

Es ist nicht vorgesehen, personenbezogene Daten an Dritte weiterzugeben. Technische Dienste wie Firebase können zur Authentifizierung und Datenspeicherung eingesetzt werden.

6. Rechte der Nutzer

Nutzer können Auskunft, Korrektur oder Loeschung ihrer Daten verlangen. In dieser Lernanwendung erfolgt dies über das Projektteam.

Privacy Policy

1. Responsible party

The Join project team is responsible for this learning application. The application is developed as part of an educational project.

2. Data that is processed

Registration asks for a name, an email address and a password. The password must not be stored directly in Firestore and must be handled by the authentication provider.

3. Purpose of processing

The data is used to create a user account, identify the signed-in user and display the user's name inside the app.

4. Storage

User-related data is stored only as long as it is needed for using the app or for testing purposes in the project.

5. Sharing data with third parties

Personal data is not intended to be shared with third parties. Technical services such as Firebase may be used for authentication and data storage.

6. User rights

Users can request information, correction or deletion of their data. In this learning application, requests are handled by the project team.
